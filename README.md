# hey-there-demo

Small demo snippets in:
- Python (`python/`)
- TypeScript (`typescript/`)
- Go (`go/`)

## Run

### Python
```bash
python python/hello.py
```

### TypeScript
```bash
npm i -D typescript
npx tsc -p typescript/tsconfig.json
node typescript/dist/index.js
```

### Go
```bash
go run go/main.go
```
